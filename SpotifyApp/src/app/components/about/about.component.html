About This Project

This project is focused on learning Angular and TypeScript, integrating external and custom-built APIs, and building a structured and scalable application.

Technologies Used

  • ✅ Angular & TypeScript: Component-based architecture, clean file structure.
  • ✅ Spotify API: Fetching artist data, albums, tracks, and obtaining API tokens.
  • ✅ Node.js & Express: Custom backend API for managing favorite tracks.
  • ✅ MongoDB: Storing and retrieving favorite tracks efficiently.

Project Features

  • 🎵 Search and browse artists and their albums.
  • 🎶 View album details and track listings.
  • ⭐ Add and manage favorite tracks.
  • 🚀 Clean and modular file structure using components, services, and models.
